## PikPak 사용법 Step 2: 앱 & 확장 프로그램 설치

### PC 사용자라면 브라우저 확장 프로그램 필수

크롬 또는 엣지 브라우저에서 **PikPak Browser Extension**을 설치하면 웹페이지에 있는 마그넷 링크와 토렌트 파일을 자동으로 감지해줍니다. 마그넷 링크 옆에 픽팍 아이콘이 뜨고, 클릭 한 번이면 내 클라우드로 바로 전송됩니다.

설치 방법:
1. Chrome 웹 스토어 또는 Edge 확장 스토어에서 "PikPak" 검색
2. "PikPak Browser Extension" 설치
3. 확장 아이콘 클릭 → 로그인

이게 설치되어 있으면 토렌트 사이트에서 마그넷 링크 옆에 픽팍 다운로드 버튼이 자동으로 붙습니다. 진짜 편리합니다.

iOS는 앱스토어 정책 때문에 앱 내 오프라인 다운로드 기능이 제한됩니다. 대신 **웹 브라우저(mypikpak.com)** 나 **텔레그램 봇(@PikPak_Bot)** 을 통해 동일한 기능을 이용할 수 있습니다.

## PikPak 사용법 Step 3: 실제로 다운로드해보기

픽팍에서 파일을 저장하는 방법은 크게 네 가지입니다.

### 방법 1: 마그넷/토렌트 링크로 클라우드 다운로드

가장 핵심 기능입니다. 웹이나 앱에서 "+" 버튼 클릭 → "링크 다운로드" 선택 → 마그넷 링크 또는 토렌트 링크를 붙여넣으면 끝입니다. PikPak 서버 캐시에 동일한 파일이 있으면 수초 안에 완료됩니다.

> **PikPak 서버는 토렌트 창고라고 생각하면 이해가 쉽습니다.** 마그넷 링크는 주문서고, PikPak 서버가 주방 역할을 합니다. 이미 캐시된 음식이라면 바로 나옵니다.

### 방법 2: 소셜 미디어 링크 저장

X(트위터), 틱톡, 페이스북에 올라온 영상 링크를 복사해서 PikPak에 붙여넣으면 자동으로 영상을 내려받아 저장해줍니다. 마음에 드는 쇼츠 영상을 영구 보관하고 싶을 때 유용합니다.

### 방법 3: 텔레그램 봇 활용

텔레그램에서 **@PikPak_Bot** 을 추가하고 픽팍 계정과 연동하면, 봇에게 마그넷 링크나 파일을 전송하는 것만으로 자동으로 내 픽팍 클라우드에 저장됩니다. 텔레그램 채널에 있는 영상 파일도 봇을 통해 바로 픽팍에 보낼 수 있습니다.

### 방법 4: 직접 파일 업로드

로컬에 있는 파일을 직접 올리는 것도 됩니다. 웹 버전은 파일당 4GB, PC 클라이언트는 파일당 20GB 제한이 있으며 하루 최대 1,000개 파일을 업로드할 수 있습니다.

## PikPak 사용법 Step 4: 저장된 파일 보기 & 스트리밍

파일이 클라우드에 저장되면 어디서든 접근할 수 있습니다. 특히 영상 파일은 **원본 화질 4K 스트리밍**이 가능합니다. 유튜브처럼 바로 재생됩니다.

다양한 해상도 옵션도 자동으로 제공되어 네트워크 상태에 따라 맞춰서 볼 수 있습니다. WebDAV를 지원하기 때문에 rclone, CloudDrive2, Alist 같은 도구와 연동하면 PC 탐색기에서 가상 드라이브로 마운트해서 쓸 수도 있습니다. VR 헤드셋을 통해 4K VR 영상을 실시간으로 스트리밍하는 사용자들도 있습니다.

Stremio + MediaFusion 애드온을 통해 PikPak과 연동하면 영화 검색, 토렌트 검색, 자막 검색을 한 번에 해결하는 고급 활용도 가능합니다.

## PikPak 단점도 알고 쓰자

솔직하게 쓰는 겁니다. 장점만 있는 서비스는 없으니까요.

- **무료 사용자 저장 공간 6GB**: 구글 드라이브 15GB보다 적습니다. 진지하게 쓰려면 프리미엄 필수
- **iOS 앱 오프라인 다운로드 제한**: 앱스토어 정책 때문에 앱에서 직접 오프라인 다운로드가 안 됩니다. 웹 브라우저나 텔레그램 봇으로 우회 가능
- **macOS 클라이언트 없음**: 현재 개발 중이라고 공식 발표. macOS는 웹 버전 사용
- **WebDAV 업로드 미지원**: 읽기/스트리밍은 되지만 WebDAV로 업로드는 안 됨
- **서비스 안정성**: 2024년 9월 데이터센터 화재로 9일 이상 장애가 발생한 전례가 있습니다
- **온라인 문서 미리보기 불가**: PDF, DOCX 같은 문서 파일은 온라인 뷰어 미지원
- **로컬 파일 동기화 없음**: 구글 드라이브처럼 폴더 자동 동기화 기능은 없습니다
